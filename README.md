# 💼 JobTracker - Front-end

Interface web do **JobTracker**, uma aplicação para organizar candidaturas, entrevistas e oportunidades de emprego.

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript

## 📌 Status

- ✅ Interface inicial concluída
- 🔄 Integração com API ASP.NET Core em desenvolvimento
- ⏳ Banco de dados MySQL e Entity Framework (próxima etapa)
